Parámetros.

Utilizaremos el dataset de IRIS proporcionado por UCI.
Se trata de una coleccion de datos recolectados sobre el genero de plantas "Iris", el mismo esta compuesto por 150 registros divididos en 3 clases de planta distintos, por cada clase existen 50 registros con los siguientes parámetros.

Sepal length

La longitud del sépalo medida en centímetros.

Sepal width

El ancho del sépalo medido en centímetros.

Petal length

La longitud del pétalo medida en centímetros.

Petal width

El ancho del pétalo medido en centímetros.

Class

La clase Iris a la que pertenece la planta.
-Iris Setosa
-Iris Versicolour
-Iris Virginica

Datos estadísticos.

El primer recurso que utlizamos es una planilla electrónica típica en la que podemos calcular datos como la media y la desviación estandar para cada uno de los parámetros del dataset, en este caso utilizamos un complento que proporciona excel para obtener varios datos estadisticos, pero tambien podriamos calcularlos de forma manual.

Importando datos.

Es importante prestar mucha atencion a la codificación de los datos así como el formato que tienen originialmente, en este caso el dataset está separado por cómas y el separador decimal es un punto por lo tanto necesitamos efectuar varias transformaciones para que los datos no queden en una unidad incorrecta.

Obtenemos la información.

Luego de las transformaciones podemos obtener las estadísticas para cada parámetro.

Otras Herramientas.

Existen muchas herramientas para trabajar con nuestros datasets, a continuación realizaremos los cálculos estadísticos utilizando RAPIDMINER.

Nuevo proyecto con dataset iris.

Generamos un nuevo proyecto en blanco y utilizamos el dataset de IRIS proporcionado por el propio RAPIDMINER.

Análisis de parámetros.

En la salida podemos observar que hay parámetros sin nombre y un parámetro adicional de ID el cual descartaremos ya que no aporta valor.

Renombramos los parámetros y descartamos la id utilizando los operadores Rename y Select Attributes.

Observamos los cambios que realizamos en el dataset en la salida así como la distinción del parametro objetivo(la clase en color verde) .

Finalmente observamos los datos estadísticos los cuales coinciden con los calculados en la planilla electrónica.

Si estamos interesados en mas datos estadísticos podemos utilizar el operador Aggregate que nos permite obtener datos más particulares de cada parámetro.
