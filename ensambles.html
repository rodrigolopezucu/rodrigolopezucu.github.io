ensambles Bayes

Algoritmo de ensambles Bayes aplicado al dataset de golf.

Recursos

  • Golf Scoring

Problema.

El problema consiste en predecir la clase de iris que tiene una planta .

Utilizaremos el dataset de IRIS proporcionado por UCI.
Se trata de una coleccion de datos recolectados sobre el genero de plantas "Iris", el mismo esta compuesto por 150 registros divididos en 3 clases de planta distintos, por cada clase existen 50 registros con los siguientes parámetros.

Sepal length

La longitud del sépalo medida en centímetros.

Sepal width

El ancho del sépalo medido en centímetros.

Petal length

La longitud del pétalo medida en centímetros.

Petal width

El ancho del pétalo medido en centímetros.

Class

La clase Iris a la que pertenece la planta.
-Iris Setosa
-Iris Versicolour
-Iris Virginica

Votación.

Se basa en aplicar distintos algoritmos y realizar una votación para lograr el mejor resultado

A continuación utilizamos el operador "Split Validation" para separar el dataset en entrenamiento y scoring y poder medir la performance de método que aplicaremos.

Dentro del operador Validation agregamos el operador "Vote" que aplicara varios modelos y generara una votación para obtener el mejor resultado, del lado derecho aplicaremos el modelo con el operador Apply Model y mediremos la performance con el operador Performance para clasificación

Dentro del operador Vote agregamos los algoritmos que utilizaremos

Ejecutamos el proyecto y en la salida podemos observar los resultados de los modelos y el cuadro de performance

Podemos observar una medida de accuracy muy alta del 97.78%, solo falla en 1 ocasión.

Bagging.

Se aplica boostraping y agregation, a partir de un conjunto de datos se generan n conjuntos aleatorios con repeticion, luego se les aplica el modelo y finalmente se consolida el resultado

A continuación utilizamos el operador "Split Validation" para separar el dataset en entrenamiento y scoring y poder medir la performance.

Dentro del operador Validation agregamos el operador "Bagging" , del lado derecho aplicaremos el modelo con el operador Apply Model y mediremos la performance con el operador Performance para clasificación

Dentro del operador Bagging agregamos el algoritmo que utilizaremos

Ejecutamos el proyecto y en la salida podemos observar los los 10 modelos que se generaron en base al parámetro iterations del operador bagging

Podemos observar una medida de accuracydel 91.11%, menor que el obtenido con el método anterior.
